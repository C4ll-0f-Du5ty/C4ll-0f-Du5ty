### ⚡ The Mission

As the **Founder & Head of Engineering** at [Autronic Solutions](http://autronicsolutions.com/), I bridge industrial shop-floor telemetry with intelligent business workflows.

My flagship project **FIST** is a modular MES+ERP platform designed to drastically reduce factory downtime through real-time OEE monitoring, production intelligence, and seamless inventory management.

- 🔭 **Current Focus:** Scaling FIST and shipping AI-powered features in **Plan_B_Plate**
- 🛠️ **Expertise:** Django + React ecosystems, Kubernetes, scalable backend architectures
- 🎓 **Background:** BSc Computer Science & Pure Mathematics (Al-Azhar University) + ALX Software Engineering
